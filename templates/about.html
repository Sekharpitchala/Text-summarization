About

Text summerization using Transformers

TOOLS USED

Flask,Transformer Libraries,Scikit Learn,Matplotlib

Flask: Flask is a micro web framework written in Python. It is used to handle web requests, route URLs, and render templates. flask library is used to create the web application, handle requests, and manage the application's routes. Transformers

(Hugging Face): The transformers library from Hugging Face provides state-of-the-art pre-trained models for various natural language processing (NLP) tasks, including text summarization. pipeline("summarization") is used to initialize a pre-trained text summarization model.

Scikit-learn: sklearn (Scikit-learn) is a machine learning library for Python. It is used for data analysis and machine learning tasks. TfidfVectorizer is used to convert text data into TF-IDF vectors. cosine_similarity is used to calculate the similarity between two pieces of text based on their TF-IDF vectors.

Matplotlib: matplotlib is a plotting library for Python. It is used to create static, interactive, and animated visualizations in Python. Used to create bar and scatter plots for visualizing the similarity scores between the original and summarized text.

Hugging Face Transformer Libraries

pipeline

pipeline is a high-level, user-friendly API designed to streamline the use of pre-trained models for various natural language processing (NLP) tasks. It abstracts the complexity of handling different models, tokenizers, and inference processes, allowing users to focus on applying the models rather than dealing with the underlying details.
